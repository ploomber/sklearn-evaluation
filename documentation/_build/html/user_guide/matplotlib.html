Advanced usage using matplotlib#

In [1]: from sklearn.ensemble import RandomForestClassifier

In [2]: from sklearn.model_selection import train_test_split

In [3]: from sklearn import datasets

In [4]: from sklearn_evaluation import plot

In [5]: import matplotlib.pyplot as plt

In [6]: data = datasets.make_classification(n_samples=200, n_features=10,
   ...:                                     n_informative=5, class_sep=0.7)
   ...: 

In [7]: X = data[0]

In [8]: y = data[1]

# shuffle and split training and test sets
In [9]: X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=.5,
   ...:                                                     random_state=0)
   ...: 

In [10]: est = RandomForestClassifier(n_estimators=10)

In [11]: est.fit(X_train, y_train)
Out[11]: RandomForestClassifier(n_estimators=10)

In [12]: y_true = y_test

In [13]: y_score = est.predict_proba(X_test)

In [14]: est = RandomForestClassifier(n_estimators=3)

In [15]: est.fit(X_train, y_train)
Out[15]: RandomForestClassifier(n_estimators=3)

In [16]: y_score2 = est.predict_proba(X_test)

As we mentioned in the previous section, using the functional interface provides great flexibility to evaluate your models, this sections includes some recipes for common tasks that involve the use of the matplotlib API.

Saving plots#

In [21]: ax = plot.roc(y_true, y_score)

In [22]: fig = ax.get_figure()

In [23]: fig.savefig('my-roc-curve.png')

Comparing several models with one plot#

In [24]: fig, ax = plt.subplots()

In [25]: plot.roc(y_true, y_score, ax=ax)
Out[25]: <AxesSubplot: title={'center': 'ROC'}, xlabel='False Positive Rate', ylabel='True Positive Rate'>

In [26]: plot.roc(y_true, y_score2, ax=ax)
Out[26]: <AxesSubplot: title={'center': 'ROC'}, xlabel='False Positive Rate', ylabel='True Positive Rate'>

In [27]: ax.legend(['Model 1', 'Baseline', 'Model 2'])
Out[27]: <matplotlib.legend.Legend at 0x13250da80>

In [28]: fig
Out[28]: <Figure size 640x480 with 1 Axes>
